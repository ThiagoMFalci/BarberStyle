# BarberStyle - Site de Barbearia

## 📌 Sobre o projeto

BarberStyle é uma aplicação web para barbearias, desenvolvida com frontend em React e backend em C#.

O sistema permite que clientes conheçam os serviços oferecidos, visualizem informações da barbearia e realizem agendamentos de forma prática. Além disso, a API possibilita o gerenciamento de serviços, profissionais, horários e reservas.

## 🚀 Tecnologias utilizadas

### Frontend
- React
- JavaScript
- HTML5
- CSS3
- Axios
- React Router DOM

### Backend
- C#
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server

## ✨ Funcionalidades

- Página inicial da barbearia
- Listagem de serviços
- Exibição de preços
- Página de profissionais
- Galeria de cortes
- Formulário de contato
- Sistema de agendamento
- API para cadastro e gerenciamento de dados
- Integração entre frontend e backend
